Reflexão e Aplicação Prática

Lista de propriedades CSS utilizadas:

Propriedade: "justify-content"

  • Flex-start
  • Flex-end
  • Center
  • Space-between
  • Space-around

Propriedade: "align-items"

  • Flex-start
  • Flex-end
  • Center
  • Basekine
  • Stretch

Aplicação Prática: Menu de Navegação

Para o Menu de Navegação, iremos dividir em 3 itens, como: Início, Sobre,Destaques. Tudo isso no topo da página de navegação. Nesse sentido colocaremos os itens do menu em uma "div", usaremos o "display:flex" para alinhar os mesmos.Além disso,utilizaremos o "align-items: center" para alinhá-los e o "justify-content: Space-around ou Space-between" para que o haja espaço distribuido entre eles.
